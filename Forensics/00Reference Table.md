### **Windows Forensic Artifacts - Quick Reference Table**

| **Artifact Name**                                                    | **Purpose**                                      | **Extraction Method**                                              | **Location**                                                                                                         |
| -------------------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| **Prefetch Files**                                                   | Tracks recently executed applications.           | Copy `.pf` files, analyze with `PECmd`.                            | `C:\Windows\Prefetch\`                                                                                               |
| **ShimCache (AppCompatCache)**                                       | Tracks executed applications.                    | Extract from SYSTEM hive with `RegRipper`, `AppCompatCacheParser`. | `HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\AppCompatCache`                                               |
| **Amcache.hve**                                                      | Tracks executed programs, hashes, install paths. | Extract `Amcache.hve` with `AmcacheParser`.                        | `C:\Windows\AppCompat\Programs\Amcache.hve`                                                                          |
| **Windows Event Logs (EVTX)**                                        | Logs system events, logins, app activity.        | Extract `.evtx` files, analyze with `Event Viewer`, `EvtxECmd`.    | `C:\Windows\System32\winevt\Logs\`                                                                                   |
| **Registry Hives**                                                   | Stores system/user configurations.               | Extract hives, analyze with `RegRipper`, `RegistryExplorer`.       | `C:\Windows\System32\config\`, `C:\Users\*\NTUSER.DAT`                                                               |
| **Jump Lists**                                                       | Tracks recently accessed files/applications.     | Analyze with `JumpLister`, `JLParser`.                             | `C:\Users\*\AppData\Roaming\Microsoft\Windows\Recent\AutomaticDestinations\`                                         |
| **Recycle Bin ($Recycle.Bin)**                                       | Stores deleted files before permanent deletion.  | Extract `$I` and `$R` files, parse with forensic tools.            | `C:\$Recycle.Bin\SID\`                                                                                               |
| **LNK (Shortcut) Files**                                             | Stores file access history.                      | Extract `.lnk` files, analyze with `LNKParser`.                    | `C:\Users\*\AppData\Roaming\Microsoft\Windows\Recent\`                                                               |
| **Shellbags**                                                        | Tracks folder access and navigation history.     | Extract `NTUSER.DAT`, parse with `ShellBagsExplorer`.              | `HKCU\Software\Microsoft\Windows\Shell\BagMRU`                                                                       |
| **UserAssist**                                                       | Tracks GUI-based program execution.              | Extract `NTUSER.DAT`, parse with `UserAssistView`.                 | `HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist`                                                 |
| **SRUM (System Resource Usage Monitor)**                             | Tracks app usage and network activity.           | Extract `SRUDB.dat`, parse with `srumecmd`.                        | `C:\Windows\System32\sru\SRUDB.dat`                                                                                  |
| **MFT (Master File Table)**                                          | Tracks file metadata and changes.                | Extract from disk, parse with `MFTECmd`, `AnalyzeMFT`.             | `$MFT` in NTFS file system.                                                                                          |
| **USN Journal ($J)**                                                 | Tracks file system changes (creation, deletion). | Extract from disk, parse with `MFTECmd`.                           | `$Extend\$UsnJrnl`                                                                                                   |
| **BitLocker Recovery Keys**                                          | Stores encryption recovery keys.                 | Extract from Registry, AD, or Microsoft Account.                   | `C:\Users\*\AppData\Local\Microsoft\Recovery\`                                                                       |
| **Windows Search Index**                                             | Tracks indexed file content and metadata.        | Extract `Windows.edb`, parse with `esedbexport`.                   | `C:\ProgramData\Microsoft\Search\Data\Applications\Windows\Windows.edb`                                              |
| **Pagefile.sys**                                                     | Stores memory pages when RAM is full.            | Extract file, analyze with `Volatility`.                           | `C:\pagefile.sys`                                                                                                    |
| **Hiberfil.sys**                                                     | Stores RAM contents during hibernation.          | Extract file, analyze with `Volatility`.                           | `C:\hiberfil.sys`                                                                                                    |
| **SRP (Software Restriction Policies)**                              | Tracks application execution policies.           | Extract from Registry, parse with `RegRipper`.                     | `HKLM\Software\Policies\Microsoft\Windows\Safer`                                                                     |
| **Windows Defender Logs**                                            | Stores security and malware detection events.    | Extract from Event Logs and `MpCmdRun.log`.                        | `C:\ProgramData\Microsoft\Windows Defender\`                                                                         |
| **Browser Artifacts (Chrome & Edge)**                                | Tracks browsing history, cookies, cache.         | Extract `.sqlite` databases, parse with `BrowsingHistoryView`.     | `AppData\Local\Google\Chrome\User Data\Default\` (Chrome), `AppData\Local\Microsoft\Edge\User Data\Default\` (Edge). |
| **RDP Cache (Remote Desktop Protocol)**                              | Tracks remote desktop connections.               | Extract bitmap cache, analyze RDP logs.                            | `C:\Users\*\AppData\Local\Microsoft\Terminal Server Client\Cache\`                                                   |
| **USB Device History**                                               | Tracks connected USB devices.                    | Extract from Registry, parse with `USBDeview`, `RegRipper`.        | `HKLM\SYSTEM\CurrentControlSet\Enum\USBSTOR`                                                                         |
| **Network Connections (Netstat, Firewall, Sysmon, ARP, SMB, Wi-Fi)** | Tracks network activity and open connections.    | Extract using `netstat`, `pfirewall.log`, `Sysmon`, `ARP cache`.   | `Firewall logs, Sysmon logs, Wi-Fi Profiles, SMBClient logs`.                                                        |
